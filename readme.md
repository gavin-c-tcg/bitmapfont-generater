# Phaser bitmap font generator

## 安裝

- 安裝失敗 https://github.com/Automattic/node-canvas/issues/1065

### Windows, Docker

```bash
	yarn docker:build
	yarn docker:sh ## 進入 docker
	yarn install
	yarn start
```

### MAC 不推薦

```bash
	brew install pkgconfig
	brew install pixman
	brew install cairo
	brew install pango

	yarn install
	yarn start
```
